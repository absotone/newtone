# newtone
Get inspiration for your work by discovering music made by neural networks!

## Dataset
[Link to the Dataset](https://www.kaggle.com/datasets/soumikrakshit/classical-music-midi)

`pip install -r requirements.txt`

`cd src/`
`python train.py`
